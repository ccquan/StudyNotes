# HTML5 与 CSS3 基础教程

### 视频与音频标签

- audio

  ```html
  <audio src="images/music.wav" controls="true"></audio>
  ```

- video

  ```html
  <video controls="true" height="290px" width="490px">
    		<source src="images/movie.mp4" type="video/mp4">
  </video>
  ```

  

### border边框标签

border-style：

- solid  实线
- dashed 虚线
- dotted  原点线
- double  双实线
- none  无边框
- groove  根据border-color的值画3D凹槽
- ridge: 定义3D脊边框。效果取决于边框的颜色值
